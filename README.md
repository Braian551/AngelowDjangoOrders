# AngeloDjangoOrders

Aplicacion Django para gestionar clientes y pedidos con autenticacion, roles administrativos, formularios validados, plantillas HTML y base de datos MySQL mediante Docker Compose.

## Arquitectura C4

El proyecto documenta su arquitectura mediante el modelo C4:

- C1: Diagrama de contexto.
- C2: Diagrama de contenedores.
- C3: Diagrama de componentes.
- C4: Diagrama de codigo.

Los diagramas se encuentran en la carpeta `docs/uml/`:

- `docs/uml/c1_contexto.puml`
- `docs/uml/c2_contenedores.puml`
- `docs/uml/c3_componentes.puml`
- `docs/uml/c4_codigo.puml`

Tambien se incluye el analisis archivo por archivo en `docs/analisis_archivos.md`.
